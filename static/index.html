Turn Any Website into an API

A powerful REST API that runs a headless browser for you. Extract HTML, CSS, and JSON from any webpage with a simple API call. No more managing complex browser automation scripts.

Why Browser API?

Focus on your data, not on browser management.

Powerful Content Extraction

Use CSS selectors to extract exactly what you need. Get clean HTML, computed CSS, or structured JSON data from any element on the page.

Full Component Collection

Collect all the CSS and JavaScript associated with a specific component, including stylesheets, inline styles, and event listeners. Perfect for component libraries.

Simple & Fast

Built-in caching and a simple, intuitive API mean you get your data quickly and efficiently. Powered by Spring Boot and Playwright for robust performance.

Get Started in Minutes


# 1. Build the project with Maven
$ mvn clean install

# 2. Build the Docker image
$ docker build -t browser-api .

# 3. Run the container
$ docker run -p 8080:8080 browser-api:latest

Once running, the API will be available at http://localhost:8080. Check out the documentation for a full API reference.
